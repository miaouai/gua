# 🪙 铜钱爻卦

> 表为求卦，里为求心

**在线演示**: https://miaouai.github.io/gua

## 简介

基于《周易》六十四卦的铜钱起卦工具，采用**朱熹变占七法则**进行专业解卦。

## 功能特点

- ✋ **两种起卦方式**：手动点击 / 电子抛币
- 🪙 **正确的阴阳判定**：字面=阴，花面=阳
- 🔮 **本卦与变卦**：自动计算动爻变化
- 📜 **朱熹变占七法则**：专业解卦逻辑
- 🎨 **色彩分级系统**：重要性一目了然
  - 🔴 红色：核心卦辞
  - 🟠 橙色：重要卦辞
  - 🟡 黄色：参考卦辞
  - 🟢 绿色：次要卦辞
  - 🔵 蓝色：背景信息
- 🤖 **AI 智能解读**：一键跳转 DeepSeek/通义千问/豆包

## 使用方法

1. 输入所问之事
2. 选择起卦方式（手动点击或电子抛币）
3. 完成 6 次抛币
4. 查看卦象解析和色彩分级提示
5. 点击 AI 按钮获取详细解读

## 技术栈

- 纯 HTML/CSS/JavaScript 单文件应用
- 无需后端，直接部署到 GitHub Pages
- 移动端优化，支持 iOS/Android
